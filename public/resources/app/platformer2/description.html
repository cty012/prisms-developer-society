Description

Platformer 2 is a multiplayer platformer game implemented with Python 3 and PyGame. In the game, your goal is to collect coins and reach the final target. While playing, elevators and buttons will aid you while monsters and lava will try to stop you. Use your brain and carefully plan a route to pass the level! Futher challenge yourself by collecting more coins and accomplishing the goal in the shortest time possible!

System Requirements

  • Windows: Windows 10+

  • MacOS: MacOS 10.13+ (High Sierra)

  • Source code: Python 3.8+, PyGame 2.0+

Controls & Keyboard Shortcuts

  • A/←: Move the player to the left

  • D/→: Move the player to the right

  • W/↑/space: Jump

  • esc: Toggle pause (works for single player and server of multiplayer)

  • ctrl+F: Toggle fullscreen (works on Windows only)

  • enter: Confirm (works differently in different scenes)

Tutorials

The gameplay tutorials can be found in the game. Level 1~7 are single player levels and level 8~10 are multiplayer levels.

To open a multiplayer game, make sure the host (who will be the server) and the clients are connected to the same network (Dongba WIFI doesn't work).

For hosts, choose New Game → Multiplayer (located at the bottom of the screen) → Choose level.

For clients, choose Join Game → Enter the server ip (which is shown at the top of the room opened by the host).

Then the host can start the game.

Bugs

There might be some minor problems in the game (won't affect the gameplay). As long as you avoid doing anything unexpected they will not appear.
