Description

Py-Generals is a Python version of the game generals.io. In each game, each player starts with a single base on the map, explores the map, and then tries to capture other players' bases. A typical game is about 5 minutes in length. After the game is ended, players can watch the replay of the game to discuss their tactics, explore new strategies, or merely recall the excitement of conquering lands and capturing bases.

System Requirements

  • Windows: Windows 10+

  • MacOS: MacOS 10.13+ (High Sierra)

  • Source code: Python 3.8+, PyGame 2.0+

Controls & Keyboard Shortcuts

  • esc: Pause / unpause

  • ctrl+F: Toggle fullscreen (works on Windows only)

  • return: Confirm (works differently in different scenes)

The following keyboard shortcuts only work in games:

  • W/A/S/D: Move the map up / left / down / right

  • shift+W/A/S/D: Move the map to the top / left / bottom / right edge

  • ↑/←/↓/→: Move the selected grid one step to the top / left / bottom / right

  • space: Cancel all commands in the queue

  • shift+1~9: Add / remove bookmark (The base is automatically bookmarked as 1 when a game is started)

  • 1~9: Focus on bookmark

  • 0: Focus on the selected grid

The following keyboard shortcuts only work in replays:

  • space: Play / pause / restart

  • -/=: Decrease / increase speed

  • ←/→: Go to previous / next turn

  • ↑/↓: Fast backward / forward

Tutorials

The gameplay tutorials can be found in generals.io.

To open a multiplayer game, make sure the host (who will be the server) and the clients are connected to the same network (Dongba WIFI doesn't work).

For hosts, choose New Game → Multi-Player.

For clients, choose Join Game → Enter the server ip (which is shown at the top of the room opened by the host).

Then the host can start the game.

Bugs

There might be some minor problems in the game (won't affect the gameplay). As long as you avoid doing anything unexpected they will not appear.
