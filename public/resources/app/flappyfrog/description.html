Description

Flappy Frog is a game in which the player controls a "frog" to move forward and avoid obstacles.

System Requirements

  • Windows: Windows 10+

  • MacOS: MacOS 10.13+ (High Sierra)

  • Source code: Python 3.8+, PyGame 2.0+

Controls & Keyboard Shortcuts

  • left mouse button: Jump

  • A: Activate angry mode
